* Free software: MIT
* Documentation: <https://easy-file.readthedocs.io>


## Features

* based on `pathlib.Path`
* UTF-8 by default
* fast JSON serialization/deserialization with [orjson](https://github.com/ijl/orjson)
* YAML serialization/deserialization
